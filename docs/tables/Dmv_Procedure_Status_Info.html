Entity

Entity: Systems.Monitoring.ProcedureStatusInfo

Dynamic management view for long running procedures. Entity: Dmv_Procedure_Status_Info (Introduced in version 23.1.1.53)

Summary

Name Type Description
Completed bit True if the procedure is completed.
ElapsedMin nvarchar(64) The elapsed minutes of the procedure at the time of the request.
ElapsedPercent float The elapsed percent of the procedure at the time of the request.
EndTime nvarchar(64) The end time of the procedure.
Error nvarchar(128) If not null, this is the text of the exception that occured during the procedure execution.
Id uniqueidentifier The ID of the long-running procedure.
Procedure nvarchar(128) The name of the procedure.
Properties nvarchar(128) List of procedure status properties.
StartTime nvarchar(64) The start time of the procedure.
User nvarchar(128) The user started the procedure.
