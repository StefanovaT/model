Table Gen_Product_Catalogs

Entity

Entity: General.Products.ProductCatalogs

Product catalogs serve to organize the products for display primarily in web pages. Entity: Gen_Product_Catalogs

Summary

Name Type Description
Background_Color int When not NULL, specifies the background color to use for visualization of the catalog. The color is in RGBA color format.
Background_Image varbinary The background image to be used for web visualization of the catalog. NULL means that background image won't be displayed.
Background_Position_Horizontal nvarchar(1) Allowed: L, C, R Horizontal position of the Background Image. L=Left, C=Center, R=Right.
Background_Position_Vertical nvarchar(1) Allowed: T, C, B Vertical position of the Background Image. T=Top, C=Center, B=Bottom.
Background_Repeat nvarchar(1) Allowed: R, X, Y, N Specifies if and how the background image will be repeated. R=Repeat both vertically and horizontally; X=Repeat only horizontally; Y=Repeat only vertically; N=No repeat.
Footer_Html nvarchar(max) The footer of the catalog in HTML format. Primarily used for web visualizations of the catalog. NULL means that there shouldn't be any user-defined footer.
Header_Html nvarchar(max) The header of the catalog in HTML format. Primarily used for web visualizations of the catalog. NULL specifies that there shouldn't be any user-defined header.
Logo varbinary The logo of the product catalog. Used for web and other visualization of the catalog.
Notes nvarchar(max)
Product_Catalog_Code nvarchar(16) Unique catalog code.
Product_Catalog_Id uniqueidentifier PK Unique catalog id.
Product_Catalog_Name nvarchar(254) ML Product catalog name (multilanguage).
Root_Product_Group_Id uniqueidentifier The root (starting) product group of the catalog. Each catalog starts from a root product group and includes the products in the sub-groups.
Row_Version timestamp
Web_Site_Id uniqueidentifier The ECommerce web site, which will be used to host the product catalog. When NULL, the product catalog would not be hosted with internal ECommerce site.
