Table Crm_Product_Prices

Entity

Entity: Crm.ProductPrices

Specific prices of products. A price is applied after matching the specified criteria. Entity: Crm_Product_Prices

Summary

Name Type Description
Campaign_Id uniqueidentifier Тhe marketing campaign to which the current definition belongs.
Currency_Id uniqueidentifier The currency of the price
Customer_Id uniqueidentifier When not NULL, specifies that the customer of the sales document must be the specified customer.
Distribution_Channel_Id uniqueidentifier Use the price only when selling through the specified channel
Enterprise_Company_Id uniqueidentifier When not NULL, specifies that the sales document must be in the specified enterprise company.
Enterprise_Company_Location_Id uniqueidentifier When set, specifies that the sales document must be of the specified enterprise company location.
From_Date date Starting date of validity of the price
Max_Quantity decimal(18, 3) Maximum quantity for which this price is valid in the Price_Quantity_Measurement_Unit
Min_Quantity decimal(18, 3) Minimal quantity required to use this price (in the Price_Quantity_Measurement_Unit)
Notes nvarchar(254)
Price decimal(13, 5) Price in the specified currency and for the specified quantity
Price_List_Id uniqueidentifier When not NULL, specifies that the sales document must have the specified price list.
Price_Quantity decimal(10, 3) The quantity of the product for which the price is specified
Price_Quantity_Measurement_Unit_Id uniqueidentifier The measurement unit of Price_Quantity
Price_Type_Id uniqueidentifier Price type of the current product price. The price types are used to set additional priority condition for the prices.
Priority tinyint Allowed: 1, 2, 3, 4, 5 Priority of the price comparative to other prices. Only the highest priority price is applied.
Product_Id uniqueidentifier The product for which a price will be defined
Product_Price_Id uniqueidentifier PK
Row_Version timestamp
Ship_To_Customer_Id uniqueidentifier When not null, specifies that the price will be applied only when the sales document has the specified Ship-To-Customer.
Target_Group_Id uniqueidentifier When not NULL, specifies a criteria, which is matched only when the customer of the sales document is included in the group.
Thru_Date date Ending date (inclusive) of the validity of the price
