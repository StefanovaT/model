Table Sys_Form_Layouts

Entity

Entity: Systems.Core.FormLayouts

Contains user layouts of the screen forms. Entity: Sys_Form_Layouts

Summary

Name Type Description
Access_Key_Id uniqueidentifier The security access key which controls the access to the layout view
Application_Name nvarchar(64) The application, which consumes the layout.
Category nvarchar(36) The object category for which the layout is applied. Object category is usually user-defined and further divides the objects of a given object type.
Form_Layout_Id uniqueidentifier PK
Form_Name nvarchar(128) The form, for which the layout is applied.
Layout varbinary The byte storage of the layout.
Layout_Format nvarchar(1) Allowed: D, L, U The format of the data in the Layout column. Values can be: 'U' - uncompressed; 'L' - LZO compressed; 'D' - Deflate compressed.
Layout_Name nvarchar(64) The name of a named layout. Standard layouts have empty string names.
Layout_Xml nvarchar(max) Layout xml - not used.
Machine_Name nvarchar(128) The machine name - not used.
Panel_Name nvarchar(64) The visual panel, for which the layout is applied.
Role_Id uniqueidentifier The role, for which the layout is applied.
Row_Version timestamp
User_Name nvarchar(64) The user for which the layout is applied. NULL means that the layout is applied for all users.
