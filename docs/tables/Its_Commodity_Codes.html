Table Its_Commodity_Codes

Entity

Entity: Finance.Intrastat.CommodityCodes

Contains the approved commodity codes for each period from The Combined Nomenclature used within the European Union countries. They are used for reporting Intrastat and Excise. Entity: Its_Commodity_Codes

Summary

Name Type Description
Commodity_Code nvarchar(8) Product code from the Intrastat Combined nomenclature
Description nvarchar(4000) ML Description of the product code from the Intrastat Combined nomenclature
Intrastat_Commodity_Code_Id uniqueidentifier PK
Row_Version timestamp
Supplementary_Unit nvarchar(16) If not null, the related product should have additional measurement unit with the same code as specified in this field. The additional unit will be used for the purposes of the Intrastat declarations
Valid_From date The starting date of validity of this code.
Valid_To date The ending date of validity of this code.
