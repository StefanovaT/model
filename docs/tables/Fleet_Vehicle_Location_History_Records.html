Table Fleet_Vehicle_Location_History_Records

Entity

Entity: Applications.Fleet.VehicleLocationHistoryRecords

Stores information about vehicle locations. Entity: Fleet_Vehicle_Location_History_Records

Summary

Name Type Description
Current_Speed_KmH int The current speed of the vehicle in km/h. NULL means the speed is unknown.
Latitude decimal(10, 7) Latitude of the vehicle at the specified time. Latitude is the vertical angle from equator to the pole of the point. Positive values denote points north of the equator. The value is in the range [-90, 90].
Longitude decimal(10, 7) Longitude of the vehicle at the specified time. Longitude is angle from the prime meridian (in England) in east-west direction. Positive values denote east and negative - west. The value is in the range [-180, 180].
Time datetime The time, for which the location is stored.
Vehicle_Id uniqueidentifier The vehicle, whose location is being stored.
Vehicle_Location_History_Record_Id uniqueidentifier PK
