Table Gen_Resource_Cost_Rates

Entity

Entity: General.Resources.ResourceCostRates

Contains cost rates for different periods for resources. Each record is valid until the next From Date. Entity: Gen_Resource_Cost_Rates

Owner Tables Hierarchy

  • Gen_Resources
  • Gen_Resource_Groups

Summary

Name Type Description
Enterprise_Company_Id uniqueidentifier The enterprise company for which the cost is defined.
From_Date date Starting date of validity of the cost rate. The rate is valid until the next cost rate record with later date is defined.
Notes nvarchar(max)
Overtime_Rate_Per_Hour decimal(12, 4) The overtime cost of the resource per hour in the resources costing currency. The overtime rate specifies the cost of the resource, when it is used outside the normal working schedule. NULL means that there is no special overtime rate and the standard rate should be applied.
Per_Use_Cost decimal(12, 2) One-time cost for each usage of the resource in the resources costing currency.
Resource_Cost_Rate_Id uniqueidentifier PK
Resource_Id uniqueidentifier The resource for which the cost rate is defined.
Row_Version timestamp
Standard_Rate_Per_Hour decimal(12, 4) The standard cost per hour of the resource in the resources costing currency. The standard cost is applied for resource usage during the normal working hours.
