Table Sm_Billing_Cycles

Entity

Entity: Crm.Subscriptions.BillingCycles

Billing cycles contain information when to issue new bills. Entity: Sm_Billing_Cycles (Introduced in version 24.1.2.5)

Summary

Name Type Description
Billing_Cycle_Id uniqueidentifier PK
Billing_Cycle_Name nvarchar(256) ML Multi-language name of the billing cycle.
Enterprise_Company_Id uniqueidentifier Specifies the enterprise company for which the cycle will be valid. NULL means that it is valid for all enterprise companies.
Is_Active bit Indicates whether the current Billing cycle is active.
Notes nvarchar(max)
Number_Of_Months tinyint Number of months between billings.
Row_Version timestamp
