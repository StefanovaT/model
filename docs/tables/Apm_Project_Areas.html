Table Apm_Project_Areas

Entity

Entity: Projects.Agile.ProjectAreas

Area of a project. Can be applicable to a single project or all projects. Entity: Apm_Project_Areas (Introduced in version 24.1.1.68)

Summary

Name Type Description
Consider_Wip_Limit int When set, specifies the work-in-progress (WIP) limit. The limit is for number of cases, which can progress to CONSIDER state.
Description nvarchar(254) ML Description of the project area.
In_Progress_Wip_Limit int When set, specifies the work-in-progress (WIP) limit. The limit is for number of cases, which can progress to IN PROGRESS state.
Is_Active bit Specifies whether the project area is active for new projects
Primary_User_Id uniqueidentifier Specified, when there is a primary user for the area
Project_Area_Id uniqueidentifier PK
Project_Area_Name nvarchar(256) ML Multi-language name of the project area
Project_Id uniqueidentifier Specified for local project areas. NULL means that the area is global and assignable for all projects.
Ready_Wip_Limit int When set, specifies the work-in-progress (WIP) limit. The limit is for number of cases, which can progress to READY state.
Row_Version timestamp
