Table Log_Shipment_Lines

Entity

Entity: Logistics.Shipment.ShipmentLines

Represents detail lines of shipments. Each line contains the shipment of one product. Entity: Log_Shipment_Lines

Owner Tables Hierarchy

  • Log_Shipments
  • Gen_Documents
