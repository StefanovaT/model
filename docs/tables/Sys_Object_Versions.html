Table Sys_Object_Versions

Entity

Entity: Systems.Core.ObjectVersions

Contains history of saved version of the managed objects. Entity: Sys_Object_Versions

Owner Tables Hierarchy

  • Sys_Objects

Summary

Name Type Description
Object_Contents varbinary The compressed contents of the object.
Object_Id uniqueidentifier The object, for which a version is stored.
Object_Version_Id uniqueidentifier PK
Row_Version timestamp
Server_Version nvarchar(15) The version of the EnterpriseOne server, which created the version. The version should be in the form (9.9.9.9). NULL means the version is unknown.
User_Id uniqueidentifier The user, which saved the version.
Version int The version number of the version, stored in the current row, starting from 1.
Version_Time datetime The timestamp when the version was saved.
Version_Type nvarchar(1) Allowed: C, U Specifies the action, which occurred on the actual object, referred by the system object. 'C' means that with this version the object was created. 'U' means that this is an update to an existing object.
