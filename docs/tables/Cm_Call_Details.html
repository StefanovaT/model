Table Cm_Call_Details

Entity

Entity: Applications.Telephony.CallDetails

Contains call detail records. Calls are phone calls, video calls and SMS messages. Entity: Cm_Call_Details

Summary

Name Type Description
Call_Detail_Record_Id uniqueidentifier PK
Call_Type nvarchar(1) Allowed: P, V, M P=Phone; V=Video; M=Message/SMS
Call_Unique_Id nvarchar(32) The unique id of the call, as reported by the telephone central. NULL when the central did not report unique Id. Used for integration purposes
Called_Party_Id uniqueidentifier The party, which received the call. NULL when the party was not determined successfully
Called_Party_Number nvarchar(80) The voice number of the party, which received the call
Calling_Party_Id uniqueidentifier The party, which originated the call. NULL when the party was not determined successfully
Calling_Party_Number nvarchar(80) The voice number of the originating party of the call
Duration_Seconds int The duration of the call (in seconds)
External_Company_Id uniqueidentifier The company of the external party. It can be the party itself, or the parent party, whichever is company. NULL when the company cannot be determined
External_Party_Id uniqueidentifier It is either the From or the To party - depending of the direction of the call. Only calls with at least one external party participating are usually logged. NULL when the respective party was null, or when no external party participated in the call
Row_Version timestamp
Start_Time datetime The starting date and time of the call
