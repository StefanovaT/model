Table Prj_Project_Task_Resources

Entity

Entity: Projects.Classic.ProjectTaskResources

Contains the resources, required by the project tasks. Entity: Prj_Project_Task_Resources

Owner Tables Hierarchy

  • Prj_Project_Tasks

Summary

Name Type Description
Billing_Price_Per_Hour decimal(12, 5) When not null, specifies the price per hour (in the currency of the Project) of resource usage which will be used for billing. NULL means that the item will be billed in another way. This way of billing is mutually exclusive with Fixed Total Price
Billing_Total_Amount decimal(14, 2) When not NULL, specifies that this item will be billed for the specified fixed total price (in the currency of the Project). NULL means that this item will be billed in another way. This way of billing is mutually exclusive with Billing Price Per Hour.
Cost_Per_Hour decimal(12, 5) Cost per hour for the resource usage for this task (in the currency of the project).
Notes nvarchar(254)
Per_Use_Cost decimal(14, 2) One time cost for each resource usage, specified in the projects currency.
Project_Task_Id uniqueidentifier The task for which the resource is planned.
Project_Task_Resource_Id uniqueidentifier PK
Resource_Id uniqueidentifier The planned resource.
Resource_Instance_Id uniqueidentifier The concrete resource instance, which should be used. NULL when no specific resource instance is required.
Resource_Usage_Hours decimal(10, 2) The total number of resource-hours, which are planned for this task. Equals to the length of the task, multiplied by the resource usage.
Resource_Usage_Percent decimal(18, 4) The planned resource usage for this activity in percents. Values of more than 100% are allowed when more than 1 resource is required.
Row_Version timestamp
Task_Total_Cost decimal(14, 2) Total cost for this task (in the currency of the project).
