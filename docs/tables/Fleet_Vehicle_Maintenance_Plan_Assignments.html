Table Fleet_Vehicle_Maintenance_Plan_Assignments

Entity

Entity: Applications.Fleet.VehicleMaintenancePlanAssignments

Represents assignment of a maintenance plan to a vehicle. Entity: Fleet_Vehicle_Maintenance_Plan_Assignments

Summary

Name Type Description
Is_Active bit Specifies whether the plan is active.
Last_Maintenance_Mileage_Km int The mileage of the vehicle (in Kilometers), when the last maintenance of this type occurred. Should be specified for plans, which require mileage check.
Last_Maintenance_Trip_Count int The trip count of the vehicle, when the last maintenance of this type occurred. Should be specified for plans, which trip count check.
Maintenance_Plan_Id uniqueidentifier The assigned periodic maintenance type.
Notes nvarchar(max)
Starting_Date date The date on which the periodic maintenance should start.
Vehicle_Id uniqueidentifier The vehicle, to which a periodic maintenance plan is assigned.
Vehicle_Maintenance_Plan_Assignment_Id uniqueidentifier PK
