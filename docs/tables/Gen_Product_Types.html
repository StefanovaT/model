Table Gen_Product_Types

Entity

Entity: General.Products.ProductTypes

Categorization of the products by their nature. Entity: Gen_Product_Types

Summary

Name Type Description
Is_Default bit When checked specifies that this type is set by default for new products unless another type is specified in the product groups.
Is_Fixed_Asset bit 1 when assets are acquired or retired when products from this type are purchased or sold
Is_Service_Activity_Service bit 1 when the product corresponds to service used in service documents (like service activities)
Is_Serviced bit 1 when the products of this type create service agreements and can be serviced
Is_Shipped bit 1 when the products from this type are being shipped after sales.
Is_Stocked bit 1=The products of this type are stocked in a warehouse; 0=otherwise (services)
Lot_Auto_Creation bit If checked specifies that lots are automatically created by the receiving orders with which the products are received.
Product_Type_Code nvarchar(16)
Product_Type_Id uniqueidentifier PK
Product_Type_Name nvarchar(254) ML
Row_Version timestamp
