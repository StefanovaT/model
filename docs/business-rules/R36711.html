    Action Validate that:
    if (PolicyKind == LogisticUnitScreenVisibility)
    , then Value = 'always' OR 'default' OR 'never'
    Description Validates if the value filled in the Value field is valid. Currently, valid values are only 'always','default' and 'never'.
    Message The 'Value' field contains an invalid value.
    The selected value '{Value}' is invalid for a warehouse policy of kind '{PolicyKind}'.
    Valid values for warehouse policies of this kind are always, default and never.
