R33773 PropertyValue - Picture Size

Name Value
Code R33773
Entity Systems.Bpm.PropertyValues Entity
Name PictureSize
Attribute Picture
Layer Back-End
Events Commit
Record State Inserted/Updated
Priority Normal
Modify NO
Action If (Picture != null)

Validate that: Picture.Length <= 10MB
Description If the field Picture is changed (insert new file) then validate that the size of the new file(picture) is equal or less than 10MB.
Message The size of the file exceeds maximum allowed size (10 MB).
Please reduce the size of the file in the limit using compression or conversion appropriately.
Version Introduced: 2023
Revocable NO
