R28979 LogisticsDocumentLine - Valid Measurement Unit

Code R28979
Entity LogisticsDocumentLines
Name ValidMeasurementUnit
Attribute MeasurementUnit
Layer Back-End
Events Commit
Priority Normal
Modify NO
Action Validate that the measurement unit in the line meets the requirements described here - Conversion Validation Methods.
Description The current validation helps the user to select the correct measurement unit in the document lines. It would be incorrect if a product is selected and its quantity is entered in a measurement unit and there is no dimension that would help convert the quantity in the line into quantity in base measurement unit of the product.
Message "Line {LineNo} has invalid measurement unit '{MeasurementUnit.Descrioption}' for product with Part Number '{Product.PartNumber}'.

A valid unit is a unit for which the product has a defined dimension or is part of the base measurement category of the product or of a category of a unit for which the product has defined dimension.

Please, change the Measurement Unit attribute."





Sort Messages By Attribute: LineNo
