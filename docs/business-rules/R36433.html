R36433 Rental Asset - Rental Asset And Managed Asset Enterprise Company Are Equal

    Code R36433
    Entity Applications.Rental.Assets Entity
    Name RentalAssetAndManagedAssetEnterpriseCompanyAreEqual
    Layer Back-End
    Attribute EnterpriseCompany
    Events Commit
    Modify NO
    Action If Rental.Asset.EnterpriseCompany != Null AND Rental.Asset.ManageAsset != Null,
    validate that Rental.Asset.EnterpriseCompany == ManageAsset.EnterpriseCompany.
    Description The rule validates that the selected managed asset has the same enterprise company
    as the rental asset.
    Message The Enterprise Company of the rental asset '{Rental.Asset.Name}'
    differs from that of the selected managed asset '{ManagedAsset.Name}'.
    Please select a managed asset with an Enterprise Company that equals
    the Enterprise Company of the rental asset.
    Introduced In Version Introduced: v.24
    Revocable NO
