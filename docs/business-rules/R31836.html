R31836 WebHosts - Name Cannot Be Defined In My Erp Net

Name Value
Code R31836
Entity Systems.Core.WebHosts Entity
Name NameCannotBeDefinedInMyErpNet
Attribute Name
Layer Back-End
Events Commit
Modify NO
Applicable Legislations ALL // no condition needed
Action Validate that:
WebHosts.Name != *.my.erp.net
Description The web host's Name cannot contain 'my.erp.net' because the *.my.erp.net domain is reserved for system purposes. The company's unique my.erp.net address is specified for the whole database and is managed by the database administrators.
If you are creating a site for the company's unique my.erp.net address, simply leave the Host field in the Site's definition blank.
Message The web host's Name cannot contain 'my.erp.net' because the *.my.erp.net domain is reserved for system purposes.
If you want to create a site for the company's unique my.erp.net address, simply leave the Host field in the Site's definition blank.
Introduced In Version Introduced: 2021
