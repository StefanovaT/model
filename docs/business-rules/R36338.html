R36338 Task - Check User Rights On Task Creation And Editing

    Code R36338
    Entity Projects.Todo.Tasks Entity
    Name CheckUserRightsOnTaskCreationAndEditing
    Attribute Owner User
    Layer Back-End
    Events Commit
    Record State Inserted,Updated
    Priority Normal
    Modify NO
    Action Validate that if ToDo.Task.SocialGroup != NULL and the GroupMember.Role of the
    OwnerUser >=” Member” for the selected ToDo.Task.SocialGroup,
    then Task creation is allowed.
    Description This rule ensures that only users with sufficient permissions
    within the specified SocialGroup (Member or Admin) can create or edit Tasks.
    Message You do not have permission to create and edit tasks in the selected social group
    '{SocialGroup.Name}'.
    Only users with ‘Member‘ or ‘Admin’ roles are allowed to create and edit tasks.
    Version Introduced: v.24
    Revocable NO
