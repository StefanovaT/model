R33758 PropertyValue - Property Value Date Validity

Name Value
Code R33758
Entity General.PropertyValues Entity
Name PropertyValueDateValidity
Attribute PropertyValueField
Layer Back-End
Events Commit
Priority Normal
Modify NO
Action If (PropertyValueField != null && PropertyValue.Property.PropertyType == PropertyType.Date)

Validate that: DateTime.TryParse(PropertyValueField, out DateTime d) == TRUE
Description If the field 'PropertyValueField' is filled in and 'PropertyType' in the definition of the Property is Date, check if PropertyValueField is valid date.
Message The value '{PropertyValueField}' is not valid date string for custom property. The date must be in format yyyyMMdd.

Example: Today = {DateTime.Today.ToString("yyyyMMdd")}.
Version Introduced: 2023
Revocable NO
